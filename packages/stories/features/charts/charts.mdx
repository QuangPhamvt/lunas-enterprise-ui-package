# Charts

Prebuilt [Recharts](https://recharts.org) v3 dashboard charts, built on the `@customafk/lunas-ui/ui/chart`
primitives (`ChartContainer`, `ChartTooltipContent`, `ChartLegendContent`, `ChartConfig`). Import from
`@customafk/lunas-ui/features/charts`. Requires the `recharts` peer dependency.

Every chart accepts `className`, `animate` (default `true`; set `false` for deterministic tests and
snapshots), and `height` (a fixed px height; omitted falls back to an `aspect-video` ratio). Colors
default to the `--chart-1`..`--chart-8` theme tokens, cycled via `getChartColor(index)`; up/down series
(candles, volume, gain/loss) use `--success`/`--danger` via `getDirectionColor`. Formatter props
(`valueFormatter`, `dateFormatter`) default to the helpers in `utils.ts`.

## Finance & stock

### CandlestickChart

OHLC candles with high-low wicks — the standard stock price chart. Recharts has no native candlestick
type, so bodies and wicks are drawn against the chart's computed axis scales.

<Canvas of={CandlestickChartStories.Default} />

Trading volume colored by day direction, paired under a `CandlestickChart` on a shared time axis.

<Canvas of={VolumeBarChartStories.Default} />

## Shared conventions

- Every chart accepts `className`, `animate`, and `height` (see above).
- Dark mode is handled by `ChartConfig`'s `{ light, dark }` theme shape plus `ChartStyle`, not by a
  `.dark` block in `theme.css`.
- Formatter props default to the helpers in `utils.ts` (`formatCurrency`, `formatCompactNumber`,
  `formatPercent`, `formatSignedPercent`, `toDateLabel`).
